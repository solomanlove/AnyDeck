# AdbManage

AdbManage 是一个轻量级 Flutter Desktop 工具箱，面向 Android 开发和 QA 调试流程。

首版实现遵循 `FLUTTER_ADB_DESKTOP_TECH_PLAN.md`：

```text
Flutter Desktop UI
  -> Dart Process 调用 adb
  -> Dart Process 启动外部 scrcpy
  -> Riverpod 管理设备、会话和长生命周期进程
```


## 更新日志

### v0.3.0 — 2026-05-27（本次更新）

本次更新新增了多项设备管理与调试增强功能：

1. **设备去重与连接方式显示**
   - 对同一物理设备通过 USB 和无线同时连接的场景进行**自动去重**，基于硬件序列号（`ro.serialno`）合并同一设备的多条连接记录。
   - 设备列表中新增**连接方式标识**（USB / 无线 / mDNS），清晰展示每台设备的当前连接途径。
2. **重启 ADB 服务**
   - 设备管理工具栏新增**"重启 ADB 服务"**按钮，可一键执行 `adb kill-server` + `adb start-server` 完成服务重启。
3. **二维码无线配对连接**
   - 新增**"无线调试配对"**功能入口，支持两种配对方式：
     - **二维码配对**：自动生成配对二维码，手机端在"开发者选项 → 无线调试 → 使用二维码配对设备"中扫描即可完成配对。
     - **配对码配对**：手动输入配对端口和配对码进行配对连接。
4. **模拟器管理**
   - 新增**"模拟器管理"**面板，自动扫描本地已创建的 Android AVD 模拟器。
   - 支持查看模拟器**运行状态**（已停止 / 启动中 / 运行中），并提供**启动**和**停止**操作按钮。
   - 运行中和启动中的模拟器自动**置顶排序**，方便快速操作。
5. **终端调试**
   - 新增**"终端调试"**标签页，为每台设备提供完整的 ADB Shell 交互终端。
   - 支持创建**多个终端实例**（多标签页），可独立关闭和删除。
   - 内置**命令历史记录**，可通过上下方向键快速翻阅历史命令。
   - 支持**常用命令收藏**，可自定义添加、删除和管理收藏命令，点击即可快速执行。
   - 支持**一键重置默认收藏命令**，恢复内置的常用命令集合。
   - 提供**清屏**、**Ctrl+C 中断**、**重启终端**等快捷操作按钮。
6. **设备控制增强**
   - 新增**菜单键**模拟（`KEYCODE_MENU`），方便调试旧版 Android 应用。
   - 新增**通知栏展开**控制（`cmd statusbar expand-notifications`），一键下拉设备通知面板。
   - 新增**屏幕自动旋转**开关，可远程切换设备的 `accelerometer_rotation` 设置。
7. **屏幕信息增强**
   - 设备信息概览中新增**屏幕逻辑密度**显示，自动将 DPI 换算为倍率并标注密度级别（如 `3x (xxhdpi)`）。
   - 新增**屏幕刷新率**显示，通过解析 `dumpsys display` 获取设备支持的刷新率信息。
   - 修复了设备信息卡片底部**"分辨率"和"字体缩放"显示不全**的问题。

### v0.2.0 — 2026-05-27（上一次更新）

1. **应用全面汉化**
   - 将全平台（macOS、Windows、Linux、Flutter 应用）显示名称统一重命名为**"安卓手机管理"**。
   - 丰富了 UI 中文本地化字符串，确保界面文案完全汉化且符合中文使用习惯。
2. **隐藏侧边栏与顶部状态显示**
   - 在选中设备后，自动隐藏左侧边栏以释放更多内容空间。
   - 顶部导航栏动态显示当前选中的设备名称。
3. **设备管理 UI 重构**
   - 重新设计了设备管理界面，移除原先无设备选中时的空白占位状态面板。
   - 引入支持排序的设备信息表格，可展示设备标识、自定义名称、连接状态等。
   - 支持编辑修改设备昵称/别名。
   - 支持一键重新连接、断开连接及删除离线设备记录。
4. **设备信息本地持久化缓存**
   - 成功加载的设备硬件及系统属性（品牌、型号、内存、分辨率等）会被自动写入本地持久化缓存。
   - 当设备离线或再次切换加载时，能够实现零延迟即时展示。
5. **拼音支持应用过滤搜索**
   - 应用管理列表的搜索框不仅支持中文和包名搜索，还全面支持**拼音首字母**和**全拼**的过滤匹配（例如输入"wx"或"weixin"即可快速筛选出"微信"）。
6. **详细应用信息弹窗展示**
   - 点击应用列表中任一应用，可打开"应用信息"详细对话框。
   - 支持查看：包名、版本、是否系统应用、最小 SDK、目标 SDK、首次安装时间、最后更新时间、安装包大小。
   - 支持动态加载获取应用大小、数据大小及缓存大小（通过 `dumpsys diskstats` 和 `dumpsys package` 等命令解析）。
   - 支持显示并一键复制**签名 MD5 校验码**。
7. **应用安装包（APK）导出**
   - 列表操作栏新增**"导出安装包"**按钮，可一键通过 `adb pull` 将设备上已安装的 APK 安装包导出并保存到本地电脑。

## 项目结构

```text
lib/
  app/
    l10n/              # 中文本地化字符串
    router/
    theme/
  core/
    adb/               # ADB 服务封装
    apps/              # 应用管理服务
    device_actions/    # 设备操作（按键、旋转、通知栏等）
    device_info/       # 设备信息采集与持久化
    emulator/          # Android 模拟器管理服务
    files/             # 文件管理服务
    logcat/            # Logcat 日志服务
    process/           # 进程与工具路径管理
    providers/         # Riverpod 全局状态管理
    scrcpy/            # scrcpy 投屏服务
    terminal/          # ADB 终端会话管理
  features/
    dashboard/
      presentation/    # 主界面、终端标签页等
```
